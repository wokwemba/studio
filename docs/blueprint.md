# **App Name**: CyberAegis AI

## Core Features:

- AI-Powered Adaptive Training: Dynamically adjust training modules and difficulty based on user performance and risk profile, providing personalized learning paths. AI tool to make recommendations about course sequencing.
- Phishing and Attack Simulations: Conduct realistic phishing and attack simulations to assess user vulnerability and identify weaknesses in security awareness.
- Real-time Risk Scoring and Metrics: Provide real-time risk scores, completion velocity, and other key metrics displayed in a Binance-style dashboard to monitor and improve cybersecurity posture.
- AI-Generated Content and Quizzes: Automatically generate training content, quizzes, and scenarios using AI to keep the training material fresh and relevant.
- Multi-Tenant Support: Support multiple tenants (enterprises) with isolated data and configurations, allowing for scalable deployment and management.
- Detailed Client Console: Allow users to track progress, see recommended modules and compete on a leaderboard
- Admin Panel AI Insights: Surface AI insights, such as detection of especially vulnerable user groups, along with actionable recommendations.  AI acts as a tool by highlighting trends and providing courses of action.

## Style Guidelines:

- Primary color: Dark Blue (#24305E) to evoke trust and security.
- Background color: Light gray (#F0F4F8), almost white, for a clean and professional look.
- Accent color: Teal (#40A298) to draw attention to important interactive elements and highlight key metrics.
- Body font: 'PT Sans', a humanist sans-serif that is modern, readable, and professional.
- Headline font: 'Space Grotesk', a modern sans-serif, a good match with the body font, but offering some additional personality and computerized flair
- Code font: 'Source Code Pro' for displaying code snippets, when necessary.
- Use a set of simple, consistent icons focused on clarity and scannability.
- Implement a dashboard layout inspired by Binance, with clear separation of key metrics and interactive charts for at-a-glance insights.
- Employ subtle animations to highlight changes in real-time risk scores and key metrics.
